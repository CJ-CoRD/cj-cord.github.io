    What is Flaxscrip?

    Inspired by Robert Anton Wilson's Illuminatus! trilogy, Flaxscrip is an fancy new cryptocurrency under development by the CoRD Dev Team. Before actually building faxscrip we are first testing individuals features on their own block chains to work out specifics and such. You can find links to the current Testscrips in the header of this page.

    Planned features for Flaxscrip

    This is a list of planned features for Flaxscrip accompanied by the currency the feature isbeing/has been/will be tested on. Features are subject to change.

  • Scrypt Hashing Algorithm (Litecoin etc.)
  • Dead Address Decay (Decayscrip)
  • True Random Block Payouts (Randomscrip)
  • Scrambled/Untracable Transactions (Zerocoin?)
  • Time Based Retargetting (Timescrip?)
  • "Fair Launch" (Stablecoin)
  • Transaction Messages (Florincoin)
  • Integrated P2Pool Mining?
  • Smaller Blockchain?
  • Expanded GUI
  • Integrated Block Explorer (ABE?)
